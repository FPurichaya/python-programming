# Python Programming Projects
This repository contains a collection of Python projects ranging from beginner programming concepts to practical automation and API integration.

The projects are designed to build a strong foundation in Python programming while introducing object-oriented programming, automation, data processing, and REST API interactions.

## Projects
1. Project Countdown App
- A countdown application that calculates the remaining time until a specified date or event.

2. Fundamental Logic
- Understanding how programs make decisions and process data through logical conditions, using function, conditional statements (if, elif, else), looping with for and while, and exception handling using try and except

3. Day to Unit Converter
- Converts a number of days into different time units.

4. Classes and Objects
- Introduces Object-Oriented Programming (OOP) concepts in Python.

5. Automation Inventory Project
- Automation inventory data project.

6. API Request to Github Project
I- nteract with the Github REST API using Python.
